    Búsqueda Global — Localiza una dirección IP exacta en todos los recursos del sistema al mismo tiempo: VIPs, nodos LTM, Self IPs y SNATs.

    ¿Cómo usarla?

    • Escribe la dirección IP exacta que quieres localizar (por ejemplo: 10.10.1.5).
    • La búsqueda es de coincidencia exacta: 10.0.0.1 no devolverá 10.0.0.10 ni 10.0.0.100.
    • Para Self IPs con máscara de red puedes escribir solo la IP (10.0.0.1) o la notación completa (10.0.0.1/24).

    Tipos de recursos buscados

    • VIP Direcciones de destino de servidores virtuales.
    • LTM Node Nodos de los pools de balanceo.
    • Self IP Direcciones propias del equipo F5.
    • SNAT Traducciones de dirección de origen.
    Útil para identificar a qué servicio o equipo pertenece una IP desconocida.
